Voici un aperçu de mes projets réalisés dans le cadre du BTS SIO SLAM, mettant en pratique mes compétences en développement web, logiciel et gestion de base de données.

PPE 1 : Site e-commerce "GameTechPro" (PHP / MySQL)

Développement d'un site e-commerce complet avec :

  • Gestion des produits et panier
  • Inscription / connexion sécurisée (mot de passe haché)
  • Recherche AJAX et tri des articles par prix
  • Passage de commande avec simulation de paiement
  • Interface responsive et moderne (HTML / CSS / JS)
  • Base de données MySQL avec les tables produits et user
